### 3️⃣ **Agent基类架构** (`agents/base_agent.py`)

#### **统一的Agent接口**
```python
class BaseAgent(ABC):
    def __init__(self, agent_name: str, config: Optional[Settings] = None):
        self.agent_name = agent_name
        self.llm_client = get_llm_client()  # 统一的LLM客户端
        self._load_prompts()  # 自动加载提示词
        
    async def _stream_llm_response(self, messages, content_type):
        """流式处理LLM响应 - 按句子或段落发送"""
        buffer = ""
        sentence_endings = ["。", "！", "？", "\n\n", "；"]
        
        async for chunk in self.llm_client.stream_chat(messages):
            buffer += chunk
            # 检查是否到达句子结尾，优化流式输出
            if any(buffer.endswith(ending) for ending in sentence_endings):
                yield await self._emit_event(EventType.LLM_CHUNK, content_type, buffer)
                buffer = ""
```

**设计亮点**：
- **流式输出优化**：按句子边界发送，提升用户体验
- **统一事件系统**：所有Agent都使用相同的事件格式
- **错误处理与告警**：自动发送飞书告警，不阻塞业务

```python
class LLMClient:
    def __init__(self, provider: str = "openai", config: Optional[Settings] = None):
        self.provider = provider
        self.provider_config = config.get_llm_provider_config(provider)
        self.client = self._init_client()  # 根据provider初始化不同客户端
    
    async def stream_chat(self, messages: List[Dict[str, str]], **kwargs) -> AsyncGenerator[str, None]:
        """统一的流式聊天接口"""
        if self.provider == "openai":
            async for chunk in self._stream_openai_chat(messages, **kwargs):
                yield chunk
        elif self.provider == "claude":
            async for chunk in self._stream_claude_chat(messages, **kwargs):
                yield chunk
        elif self.provider == "qwen":
            async for chunk in self._stream_qwen_chat(messages, **kwargs):
                yield chunk
```

**设计亮点**：
- **统一接口**：不同LLM提供商使用相同的调用方式
- **参数过滤**：自动过滤不支持的参数，避免API错误
- **错误处理**：统一的异常处理和重试机制
