# 自定义注解认证与鉴权相关内容总结
1. **项目背景与需求**：
    - 确保仅授权用户可访问特定服务接口。
    - 对内部服务间的调用进行认证，防止未授权的内部请求。
2. **自定义注解使用的元注解及作用**：
    - `@Retention(RetentionPolicy.RUNTIME)`：保证注解在运行时依然可用。
    - `@Target(ElementType.METHOD)`：限制注解只能用于方法上。
3. **AOP相关内容**：
    - 基本概念：允许在不改变业务逻辑代码的情况下添加横切关注点，如日志记录和安全检查。
    - 主要组成部分：切面(Aspect)、切点(Pointcut)、通知(Advice)和连接点(JoinPoint)。
4. **新增鉴权方式的扩展方法**：
    - 新增一个实现鉴权接口的类。
    - 在切面中增加相应的逻辑。
