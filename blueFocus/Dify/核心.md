Dify 是一个开源的 LLM 应用开发平台，具有七个核心功能模块。

## 主要功能详解

### 1. 工作流程 (Workflow)
在可视化画布上构建和测试强大的 AI 工作流程，支持拖拽式节点组合来实现复杂的业务逻辑。 [1](#0-0) 

### 2. 全面的模型支持
无缝集成数百个专有和开源 LLM 模型，支持来自数十个推理提供商和自托管解决方案，涵盖 GPT、Mistral、Llama3 以及任何与 OpenAI API 兼容的模型。 [2](#0-1) 

### 3. Prompt IDE
提供直观的界面用于制作提示词、比较模型性能，并可为基于聊天的应用程序添加文本转语音等附加功能。 [3](#0-2) 

### 4. RAG 管道
广泛的 RAG 功能，涵盖从文档摄入到检索的全流程，开箱即用地支持从 PDF、PPT 和其他常见文档格式中提取文本。 [4](#0-3) 

### 5. Agent 智能体
可以基于 LLM 函数调用或 ReAct 定义 Agent，并为 Agent 添加预构建或自定义工具。Dify 为 AI Agent 提供了 50 多种内置工具，如谷歌搜索、DALL·E、Stable Diffusion 和 WolframAlpha 等。 [5](#0-4) 

### 6. LLMOps
随时间监视和分析应用程序日志和性能，可以根据生产数据和标注持续改进提示词、数据集和模型。 [6](#0-5) 

### 7. 后端即服务 (Backend-as-a-Service)
所有 Dify 的功能都带有相应的 API，因此可以轻松地将 Dify 集成到自己的业务逻辑中。 [7](#0-6) 

## 功能对比优势

与其他平台相比，Dify 在功能完整性方面具有明显优势，支持工作流程、Agent、RAG 引擎、可观测性和企业级功能等全方位能力。 [8](#0-7) 

**Notes**
